EJERCICIO 1 
Usando como base el ejemplo que hemos utilizado en clase realiza los siguientes
apartados:
1. Guarda en la carpeta donde tienes la página web los archivos gif, png y jpeg
que te adjunto en el aula virtual.
2. Incluir un gif animado de un programador que en caso de no poder mostrarlo el
navegador se muestre como texto alternativo “Informático programando”,
modifica el tamaño del gif para que sea de 200x200
3. Insertar una imagen de la pantalla de un ordenador, modifica el tamaño de la
imagen para que sea de 400x300
4. Haz uso de las etiquetas <figure> <figcaption> para agrupar ambos contenidos
así como para incluir una leyenda de ambos con el texto “Imágenes estáticas de
Página Web formatos png y jpg, Gif animado de Informático programando”
5. Modifica los colores tal y como se muestran en la imagen, rosa para la página,
naranja barra de navegación,amarillo para la barra lateral, amarillo luminoso
para el body, azul para el contenido y dentro de este verde luminoso para cada
uno de los ejercicios, y para el pie de la página web azul luminoso
6. Busca información y explica que tipos de archivos de imagen permiten
transparencia y cuales no, así como los colores con los que trabajan estos
formatos
7. Explica todo lo que has hecho.
